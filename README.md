Nuxt.js 构建项目
=========================================
### 用法

1. 安装依赖包

   `npm install`

2. 运行开发环境

   `npm run dev`

3. build前端代码

    `npm run build`
4. 运行生产环境

    `npm start`

### gogs git钩子设置

钩子名称：`post-receive`

钩子文本： 

```shell
#!/bin/bash
unset $(git rev-parse --local-env-vars);
cd /usr/share/nginx/vue-blog
git pull origin master &&  npm i && npm run build
```
